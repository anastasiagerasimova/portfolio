О проекте

Учебный макет для приложения Юнит. Модульный подход при проектировании страницы. Применены различные техники для максимального качества вёрстки, а также добавлены ненавязчивые hover-эффекты. Активно использована svg графика. Адаптивная верстка под планшеты и смартфоны, без использования bootstrap.

Детали

  • Клиент:
    ООО “ЮНИТ”
  • Проект:
    Лендинг для приложения “ЮНИТ”
  • Верстка:
    Адаптивная верстка, HTML5, CSS3, JavaScript, jQuery.
  • Особенности:
    Модель Flexbox, позиционирование, БЭМ-нейминг методология, Mobile first подход, препроцессор LESS, карусель OwlCarousel.
