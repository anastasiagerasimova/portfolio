О проекте

Учебный проект. Многостраничный сайт для туристичекой компании “ФорестТревел”. Блочная верстка. Float и clear для блочной верстки. Cоздание разметки веб-страниц основываваясь на свойстве float и clear. Построения структуры страницы на примере основных структурообразующих блоков. Верстка сайта согласно макету с использованием Pixel Perfect.
